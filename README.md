## 概要
・Diary App (Pure PHP)
Laravel版からフレームワークを使わずに再実装したプロジェクト
https://github.com/takeC00/diary-app-laravel
(laravel版リポジトリ)

## 機能一覧

### 認証機能
- ユーザー登録
- ログイン
- ログアウト

### 日記機能（ログインユーザー）
- 自分の日記一覧
- 日記作成
- 日記編集
- 日記削除

### 公開機能
- 公開日記一覧
- 公開日記詳細表示

## 技術
- PHP（フレームワークなし）
- MariaDB
- Apache（XAMPP想定）
- AWS（EC2 / RDS）※予定

## 目的
- Laravelをはじめフレームワークを使用せず、一つ一つ自らコーディングすることで学習効果を高める。
- フレームワークを使わずに実現する方法を学ぶ。
